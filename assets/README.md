# Assets

Shared assets used across the Intune Labs organization.

## Structure

- `branding/` - Logos, brand materials, and visual identity assets
  - SVG and PNG logo variants
  - Business cards and promotional materials

## Usage

Reference assets using relative paths from your workspace:
```
../../assets/branding/intune-labs-logo.svg
```

Avoid duplicating assets across projects - use references instead.
