# Documentation

This directory contains all project documentation.

## Structure

- `deployment/` - Deployment guides and infrastructure documentation
- `presentations/` - Presentation materials and slides
- `adr/` - Architecture Decision Records (ADRs)
- Root level files:
  - `init.md` - Quick visual summary of system architecture
  - `comprehensive-analysis.md` - Detailed codebase analysis

## Guidelines

- Follow the ADR template for architectural decisions
- Keep deployment guides up-to-date with infrastructure changes
- Use Conventional Commits when updating documentation
